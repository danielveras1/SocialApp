# SocialApp
Projeto com foco em Autenticação e Autorização

## Funcionalidades: 
 - Login e Cadastro
 - Visualizar o Me
 - Alterar senha
 - Middleware de Autenticação
 - Bearer Token
 
 #### Ferramentas:
  - Linguagem TypeScript
  - Node.js
  - Express
  - TypeORM
  - bcrypt.js
  - JSON Web Token
  
 #### Links de referência:
  - https://jwt.io/
  - https://typeorm.io/
  - https://expressjs.com/pt-br/
